This application presents a digital scorecard for players of the physical Pass the Pigs game. Users can create a scorecard with 1-4 players and desired amount of rounds and easily keep score during the gameplay. 
When the match is finished, the winner is declared and user is given the option to save the match information to a local file. The saved matches can be viewed in the applicationâ€™s Match Archive.

Functionality:
* Localization
*	Graphics
*	Tile
*	Modern UI design
*	Lifecycle handling
*	MVVM architecture
*	Read/write files to disk
*	Support for different screens and themes

Missing functionality:
*	Cloud storage
